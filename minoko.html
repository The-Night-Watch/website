Minoko UI

Minoko UI (named after the character from the video game Project Eden) was a design system and Vue.js UI components library. It has since been deprecated. The code is still available but not maintained.

I have always been fascinated by design systems and branding standardisation. Minoko drew inspiration from International Typographic Style, specifically Otl Aicherâ€™s work on the 1972 Munich Olympics.

While Vue.js was a promising technology in 2016, the front-end development world has been plagued by over engineering and trend-based decision making those past few years, and I have since moved away from javascript frameworks, prefering more minimal solutions.
